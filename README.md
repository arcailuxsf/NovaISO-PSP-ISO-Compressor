# NovaISO

[Español] | [English]

---

## 🇪🇸 Descripción (Español)

**NovaISO** es una herramienta de alto rendimiento diseñada para la gestión de archivos de juegos de PSP (PlayStation Portable). Permite comprimir imágenes ISO al formato CSO (Compressed ISO) y viceversa, garantizando la máxima compatibilidad con emuladores como PPSSPP y hardware real.

### Características Principales:
- **Compresión Inteligente:** Implementa algoritmos avanzados para evitar la corrupción de archivos, asegurando una alineación de sectores perfecta.
- **Interfaz Moderna:** Diseño oscuro premium con visualización de metadatos en tiempo real (Título, ID, Región e Icono).
- **Soporte CSO Completo:** Extrae información directamente de archivos comprimidos sin necesidad de descompresión previa.
- **Sin Dependencias:** Ejecutable único y ligero escrito íntegramente en C++.

### Agradecimientos y Créditos:
Este proyecto utiliza y se inspira en las siguientes tecnologías y fuentes:
- **Zlib:** Para los algoritmos de compresión/descompresión DEFLATE.
- **ISOCompressor / MaxCSO:** Referencia técnica para la estructura del motor de compresión y compatibilidad con PSP.
- **Vibecoding:** Este proyecto fue desarrollado bajo una filosofía de *vibecoding*, priorizando la fluidez, la estética y la resolución creativa de problemas en tiempo real.

## 🇺🇸 Description (English)

**NovaISO** is a high-performance toolkit designed for managing PSP (PlayStation Portable) game files. It enables the compression of ISO images to the CSO (Compressed ISO) format and vice versa, ensuring maximum compatibility with emulators like PPSSPP and actual hardware.

### Key Features:
- **Smart Compression:** Implements advanced algorithms to prevent file corruption, ensuring perfect sector alignment.
- **Modern Interface:** Premium dark theme with real-time metadata display (Title, ID, Region, and Icon).
- **Full CSO Support:** Extracts information directly from compressed files without the need for prior decompression.
- **Standalone:** A single, lightweight executable written entirely in C++.

### Acknowledgments and Credits:
This project uses and is inspired by the following technologies and sources:
- **Zlib:** For the DEFLATE compression/decompression algorithms.
- **ISOCompressor / MaxCSO:** Technical reference for the compression engine structure and PSP compatibility.
- **Vibecoding:** This project was developed under a *vibecoding* philosophy, prioritizing flow, aesthetics, and creative real-time problem solving.
